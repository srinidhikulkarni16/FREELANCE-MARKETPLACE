# Freelance Marketplace Platform

A full-stack MERN marketplace application that allows users to create, browse, and purchase digital services.



## Features
* **User Authentication:** Secure login and registration using JWT and HTTP-only cookies.
* **Service Management:** Users can create, edit, and delete "Gigs".
* **Advanced Search:** Filter services by category, price, and sorting criteria.
* **Messaging System:** Real-time conversation capabilities between buyers and sellers.
* **Reviews & Ratings:** Comprehensive feedback system for completed services.
* **Order Tracking:** Manage active and completed transactions.

## Tech Stack
* **Frontend:** React, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB with Mongoose ODM
* **Authentication:** JSON Web Tokens (JWT) & Cookie-parser
